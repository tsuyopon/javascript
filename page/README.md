# サンプルを使う
必要なパッケージを準備します。
```
$ npm install page
$ git clone git://github.com/visionmedia/page.js
$ cd page.js
$ npm install
```

次でnodeを起動してサンプルを動作します。
```
$ node examples
```

これで簡易Webサーバとして起動しているのでpage.jsの挙動を確認できます。
```
$ open http://localhost:4000
```
